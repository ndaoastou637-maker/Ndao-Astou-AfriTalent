# AfriTalent 🌍

**Projet Semestre 2 - Groupe ISI**
Réalisé par : Ndao Astou
Classe : RT

## Description 
AfriTalent est un plateforme fictive de mise en relation entre freelances tech africains et entreprises du monde entier. Ce site vitrine présente la plateforme, ses fonctionnalités,ses tarifs et des profils de freelances.

## Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, Bento Grid, animations)
- JavaScript Vanilla (ES6+)
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (Syne, Inter)
- Git & GitHub pages

## Fonctionnalités principales
- Dark mode avec sauvegarde localStorage
- Compteurs animés au scroll (IntersectionObserver)
- Filtrage dynamique des freelances par catégorie
- Validation complète du formulaire de contact
- Navbar dynamique au scroll
- Animations fade-in des sections
- Bouton retour en haut
- Design responsive (mobile, tablette, desktop)

## Pages
- `index.html` — Page d'accueil
- `freelances.html` — Catalogue de freelances
- `tarifs.html` — Plans et tarification
- `about.html` — À propos
- `contact.html` — Contact
